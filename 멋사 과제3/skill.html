📘 1주차 - HTML 시멘틱 태그

배운 날짜: 2025년 3월 20일

📙 2주차 - CSS 폰트, 보더, 박스모델

  • 배운 기술: CSS 폰트 설정, 보더, 박스모델
  • 기술 설명: 박스 모델은 모든 HTML 요소의 레이아웃을 정의합니다. CSS를 사용해 보더, 패딩, 마진을 조정하여 웹 페이지의 디자인을 세밀하게 조정할 수 있습니다.

진행도: 80%

배운 날짜: 2025년 3월 27일
