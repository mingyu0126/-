📘 1주차 - HTML 시멘틱 태그

학습 목표

  • HTML5 시멘틱 태그 이해
  • 웹 접근성을 고려한 태그 사용
  • 웹 문서의 구조화된 작성법 익히기

HTML 시멘틱 태그는 웹 문서의 구조를 명확하게 정의하는 데 중요한 역할을 합니다. header, footer, article, section 같은 시멘틱 태그는 웹 페이지의 각 부분을 정의하고, 검색 엔진 최적화(SEO)와 웹 접근성에 큰 영향을 미칩니다.

시멘틱 태그 활용: 시멘틱 태그는 웹 콘텐츠를 더 쉽게 읽고, 관리할 수 있게 해주며, 검색엔진 최적화(SEO)에도 도움이 됩니다. header는 페이지의 상단 영역을, footer는 페이지의 하단 영역을 정의합니다. section과 article은 독립적인 콘텐츠 영역을 나타내는 데 사용됩니다.

📙 2주차 - CSS 폰트, 보더, 박스모델

학습 목표

  • CSS 폰트 스타일링과 텍스트 꾸미기
  • 보더 속성 및 여백(패딩, 마진) 다루기
  • 박스 모델 이해 및 활용

CSS 박스 모델은 웹 페이지의 모든 요소가 직사각형 박스로 나타난다는 개념을 바탕으로 합니다. 요소는 기본적으로 마진, 보더, 패딩, 콘텐츠 영역으로 나뉘며, 이들의 조합을 통해 웹 페이지의 레이아웃을 구성합니다.

보더, 패딩, 마진의 차이점을 명확히 이해하고 사용하는 것이 중요합니다. 보더는 요소의 테두리, 패딩은 내용과 테두리 사이의 간격, 마진은 요소 간의 간격을 정의합니다.

📕 3주차 - 그리드와 플렉스

학습 목표

  • 반응형 레이아웃 구현을 위한 그리드와 플렉스 학습
  • 그리드와 플렉스의 차이와 각각의 활용법
  • 다양한 화면 크기에 맞춘 레이아웃 적용 연습

그리드 시스템은 웹 페이지에서 2차원 레이아웃을 쉽게 만들 수 있게 도와줍니다. 플렉스 박스는 1차원 레이아웃을 쉽게 구성할 수 있도록 도와주며, 특히 반응형 웹 디자인에서 유용합니다.

그리드와 플렉스는 각각의 특성에 맞는 레이아웃 구성을 가능하게 하며, 그리드 시스템은 큰 레이아웃에, 플렉스는 내부 요소 배치에 주로 사용됩니다.
